What is this?

This is a site where you can upload an image of a mathematical expression and it will give you the LaTeX code that would produce that equation!

About us

Hello! We are 4 Computer Science students in the Software Development Class at Harvey Mudd College and created this web app as our project for the class.

During our time as Computer Science majors, we have had to write up homework for different classes using LaTeX, but we’ve learned that it can be a very time consuming task. Thus, we have created a website where you can upload an image and it will actually give you the LaTeX code, cutting the time spent exponentially!
